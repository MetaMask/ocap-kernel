# `@ocap/llm-bridge`

Long-running bridge process that proxies a single Anthropic LLM conversation over a Unix-socket protocol; used by the service matcher (and other LLM-backed demos) so vats can talk to the model without violating SES network restrictions

## Installation

`yarn add @ocap/llm-bridge`

or

`npm install @ocap/llm-bridge`
