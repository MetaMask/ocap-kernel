# `@ocap/demo-services`

Stub service vats for the AI-orchestration demo (LSUR pipeline): each vat registers with the matcher and produces a canned-but-spec-conformant artifact.

## Installation

`yarn add @ocap/demo-services`

or

`npm install @ocap/demo-services`

## Contributing

This package is part of a monorepo. Instructions for contributing can be found in the [monorepo README](https://github.com/MetaMask/ocap-kernel#readme).
