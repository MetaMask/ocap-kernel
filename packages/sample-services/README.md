# `@ocap/sample-services`

Standalone, MetaMask-free service vats (Echo, RandomNumber) that register with a service matcher; bundled per-vat for daemon-side launchSubcluster

## Installation

`yarn add @ocap/sample-services`

or

`npm install @ocap/sample-services`

## Contributing

This package is part of a monorepo. Instructions for contributing can be found in the [monorepo README](https://github.com/MetaMask/ocap-kernel#readme).
