# `@ocap/kernel-agents-repl`

REPL strategy for capability-enabled, language-model-flow-controlled programming.

See the [strategies documentation](../kernel-agents/src/strategies/README.md) from `@ocap/kernel-agents`

## Installation

`yarn add @ocap/kernel-agents-repl`

or

`npm install @ocap/kernel-agents-repl`
