    @ocap/kernel-agents

    Capability-enabled, language-model-flow-controlled programming.

    Installation

    yarn add @ocap/kernel-agents

    or

    npm install @ocap/kernel-agents

    Contributing

    This package is part of a monorepo. Instructions for contributing can be found in the monorepo README.

    Running E2E Tests

    The end to end tests assume an ollama server is running on localhost:11343 and has the DEFAULT_MODEL already pulled.

    Pulling an Ollama model (CLI)

    ollama pull 'llama3.1:latest'

    Pulling an Ollama model (curl) 

    curl -X POST http://localhost:11434/api/pull -d '{
      "name": "llama3.1:latest"
    }'

    Test Commands

    To run the test suite, use the yarn test:e2e command. Ollama configuration errors will be detected by the suite tests.

    To observe intermediate steps, including prompts provided to the agent, use the --no-silent flag. 

    yarn test:e2e --no-silent
