    Class Kernel

    The main class for the ocap kernel. It is responsible for managing the lifecycle of the kernel and the vats.

    Param: commandStream

    Command channel from whatever external software is driving the kernel.

    Param: platformServices

    Service to do things the kernel worker can't.

    Param: kernelDatabase

    Database holding the kernel's persistent state.

    Param: options

    Options for the kernel constructor.

    Param: options.resetStorage

    If true, the storage will be cleared.

    Param: options.logger

    Optional logger for error and diagnostic output.

    Param: options.keySeed

    Optional seed for libp2p key generation.

    Returns

    A new Kernel.
