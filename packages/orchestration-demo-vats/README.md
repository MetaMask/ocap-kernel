# `@ocap/orchestration-demo-vats`

Standalone, MetaMask-free service vats used by the orchestration demo: the stub service vats (industrial-design, schematic-generation, firmware, mechanical-design, pcb-layout, component-sourcing, device-assembly, retail-listing, logistics, bench-build), the demo wallet vat, and the Echo / RandomNumber utility vats. Bundled per-vat for daemon-side launchSubcluster.

## Installation

`yarn add @ocap/orchestration-demo-vats`

or

`npm install @ocap/orchestration-demo-vats`
